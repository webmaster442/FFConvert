FFConvert - an FfMpeg command line converter

Website: 
    https://github.com/webmaster442/FFConvert/
Issue reporting:
    https://github.com/webmaster442/FFConvert/issues
Articles in hungarian about design: 
    https://csharptutorial.hu/category/alkalmazasfejlesztes-elesben/ffmpeg-frontend/s

Usage: ffconvert [inputfiles] [preset] [outdir] [optons]

Available options:

--sh [file]
    Instead of running encode, write generated command line to specified file

This program to properly function requires a build of ffmpeg. An ffmpeg build can
be obtained from: https://ffmpeg.org/download.html
